TextEncode
==========

A Small Tool for Encoding Text. This tool is based on a small API ;)

Includes 15 popular ciphers:

01. AES128
02. AES192
03. AES256  
04. BLOWFISH  
05. CAST128  
06. CAST256  
07. DES  
08. GOST  
09. LOKI97  
10. RC2  
11. SAFERPLUS  
12. SERPENT  
13. TRIPLEDES  
14. TWOFISH  
15. XTEA  

- Written in Python 2.7
- Requires an Internet connection
